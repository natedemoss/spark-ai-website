About Spark AI

Spark AI is a lightweight Electron application that runs as a transparent, always-accessible widget on your desktop. Use it for quick calculations, research, or task assistance without disrupting your workflow.

Designed with productivity in mind, Spark AI stays in the background until you need it, then provides instant AI assistance for whatever you're working on.

Documentation

What is Spark AI?

Spark AI is an Electron-based desktop application that provides a transparent, movable chatbot widget. Built with HTML, JavaScript, and Node.js, it integrates seamlessly into your workflow, offering assistance with tasks like math, science, and research.

How It Works

The application uses Electron's BrowserWindow to create a frameless, transparent window that stays on top of other applications. Key features include:

  • Global Shortcuts: Registered using globalShortcut for toggling visibility (Ctrl+Shift+X), clearing messages (Ctrl+R), and moving the window (Ctrl+Arrow Keys).
  • IPC Communication: The main process communicates with the renderer process via ipcMain to handle window resizing and other events.
  • Dynamic Positioning: The window is centered on the screen and can be moved using arrow keys or by dragging the top bar.
  • Transparency and Focus: The window uses CSS for transparency and adjusts opacity based on focus state.

The main JavaScript file (main.js) sets up the Electron app, configures the window, and handles global shortcuts. The renderer process (in index.html) manages the UI and user interactions.

Commands

  • Ctrl + Shift + X: Toggle widget visibility
  • Ctrl + R: Clear chat messages
  • Ctrl + Arrow Keys: Move the widget
  • Drag Top Bar: Click and drag the top bar to reposition the widget

Note: Shortcuts may require the app to be in focus. Some systems may restrict global shortcuts.
